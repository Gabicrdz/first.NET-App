# ConsoleApp1 — Primer proyecto .NET

Descripción
- Proyecto de consola .NET creado como ejercicio inicial. Contiene un programa mínimo en `Program.cs` que sirve como punto de partida para aprender a compilar y ejecutar aplicaciones .NET.

Requisitos
- .NET SDK instalado (versión compatible con el TFM del proyecto).

Cómo compilar
```bash
dotnet build
```

Cómo ejecutar
```bash
dotnet run
```

Estructura del proyecto
- ConsoleApp1.csproj — archivo de proyecto.
- Program.cs — punto de entrada de la aplicación.

Contribuir
- Si quieres mejorar este proyecto, crea una rama nueva, realiza cambios y abre un pull request.

# ConsoleApp1 — First .NET project

Description
- Minimal .NET console project created as a first exercise. It contains a small program in `Program.cs` that serves as a starting point to learn how to build and run .NET applications.

Requirements
- .NET SDK installed (version compatible with the project's TFM).

Build
```bash
dotnet build
```

Run
```bash
dotnet run
```

Project structure
- ConsoleApp1.csproj — project file.
- Program.cs — application entry point.
